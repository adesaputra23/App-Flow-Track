# App-Flow-Track
Aplikasi web untuk mengelola dan memantau seluruh proses produksi rokok secara real-time. Fitur utama meliputi perencanaan bahan baku, pelacakan lini produksi, kontrol kualitas (QC), hingga laporan inventori dan efisiensi mesin. Memastikan operasional yang efisien dan kepatuhan standar industri.
